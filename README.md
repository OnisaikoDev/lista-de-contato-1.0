# 📇 Lista de Contatos 1.0

🚀 Repositório: https://github.com/OnisaikoDev/lista-de-contato-1.0  

---

## 💻 Sobre o Projeto

Este projeto é uma aplicação simples de lista de contatos desenvolvida com foco na prática de fundamentos do desenvolvimento web.

A proposta é representar visualmente uma lista de contatos, trabalhando a organização dos elementos e a construção de uma interface clara e funcional.

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  

---

## ✨ Funcionalidades

- 📋 Exibição de contatos  
- 🎨 Interface estilizada com CSS  
- 📱 Layout adaptável a diferentes tamanhos de tela  

---

## 📈 Objetivo

Este projeto foi desenvolvido com o objetivo de:

- Praticar estruturação de páginas com HTML  
- Evoluir na estilização com CSS  
- Criar interfaces organizadas e consistentes
